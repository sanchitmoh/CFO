# Utility Scripts

This folder contains utility scripts for testing and managing various services.

## Files

- `test_email.py` - Test email service functionality
- `test_slack.py` - Test Slack integration

## Usage

### Email Testing
```bash
python backend/scripts/test_email.py your-email@example.com
```

### Slack Testing
```bash
python backend/scripts/test_slack.py
```

Make sure your environment variables are properly configured before running these scripts.
